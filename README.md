# Voice Calculator

A web-based calculator that understands your voice! Speak an arithmetic expression — like *"five minus four"* or *"three times six"* — 
and the calculator will instantly compute and display the result.

## Features

-  **Voice Input** — uses the Web Speech API to capture spoken expressions in real time
-  **Natural Language Arithmetic** — understands number words like *"five"*, *"twelve"*, *"hundred"*
-  **Supports all basic operations** — addition, subtraction, multiplication, and division
-  **Clean UI** — simple, responsive interface built with HTML and CSS
-  **No backend needed** — runs entirely in the browser

##  Tech Stack

| Technology | Role |
|---|---|
| HTML | Page structure |
| CSS | Styling and layout |
| JavaScript | Speech recognition logic and calculation |
| Web Speech API | Voice input capture |

##  Browser Compatibility

This project uses the **Web Speech API**, which is best supported on:

| Browser | Support |
|---|---|
| Google Chrome |  Full support |
| Microsoft Edge |  Full support |
| Firefox |  Partial / requires flag |
| Safari |  Limited |

> For the best experience, use **Google Chrome**.

##  Notes

- Microphone permission must be granted in the browser when prompted.
- Speak clearly and at a natural pace for best recognition accuracy.
- Currently supports English language input.
